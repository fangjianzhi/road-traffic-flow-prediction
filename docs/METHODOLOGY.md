# 方法概览

本文把每 2 分钟定义为一个离散时间单位，令 `t = 0, 1, ..., 59` 对应 07:00 至 08:58。核心约束是主路监测点的流量等于各支路在传播延迟修正后的流量之和。

## 1. 双支路汇入模型

支路 1 采用严格递增的线性函数：

```text
f1(t) = a·t + b,  a > 0
```

支路 2 采用在转折点前后斜率不同的分段线性函数，主路流量满足：

```text
F(t) = f1(t) + f2(t)
```

参数估计通过观测值与模型值之间的误差最小化完成，并加入非负性、斜率方向和分段连续性约束。

## 2. 四支路汇入模型

四条支路分别对应四类典型演化规律：

- 支路 1：常数流量；
- 支路 2：增长、稳定、再次增长的分段线性流量；
- 支路 3：线性增长后稳定；
- 支路 4：正弦周期流量。

支路 1、2 到主路监测点存在一个离散单位（2 分钟）的传播时间，因此主路模型为：

```text
F(t) = f1(t-1) + f2(t-1) + f3(t) + f4(t)
```

代码利用 SciPy 的局部或全局优化器估计参数，并以 RMSE 等指标评价拟合效果。

## 3. 信号灯控制模型

对于受信号灯控制的支路，代码把红灯阶段的放行流量设为 0，并在绿灯阶段使用分段函数描述流量增长、稳定与下降。`problem_3_signal_control.py` 从竞赛附件的表 3 读取主路观测数据。

## 4. 最少采样时刻

完整观测集合共有 60 个时刻。贪心算法从边界点和模型关键点出发，每轮加入一个能够最大程度改善覆盖或预测误差的候选时刻，直至达到误差阈值或观测点数量上限。

论文给出的 12 个代表性采样时刻为：

```text
07:00, 07:02, 07:04, 07:06, 07:08, 07:10,
07:50, 08:12, 08:14, 08:16, 08:18, 08:58
```

## 5. 适用边界

模型面向城市主干道与单/双车道支路构成的标准汇入场景。当前实现没有显式建模非机动车穿行、事故、临时交通管制、节假日效应或实时参数漂移，因此不应直接用于安全关键型交通控制。
