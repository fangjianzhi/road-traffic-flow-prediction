# 数据说明

`data/competition_attachment.xlsx` 来自 2025 年第二十二届“五一”数学建模竞赛 A 题附件。工作簿包含不同道路汇入场景下，主路监测设备按 2 分钟间隔记录的车流量序列。

工作簿共有 4 个工作表，每表 60 条观测记录（另含 1 行表头）：

| 工作表 | 时间范围 | 监测变量 |
|---|---|---|
| `表1 (Table 1)` | 07:00-08:58 | 主路 3 车流量 |
| `表2 (Table 2)` | 07:00-08:58 | 主路 5 车流量 |
| `表3 (Table 3)` | 07:00-08:58 | 主路 4 车流量 |
| `表4 (Table 4)` | 07:00-08:58 | 主路 4 扩展场景车流量 |

## 使用方式

- 问题 1、2、4 的数值序列已直接写入相应脚本，便于独立运行。
- `problem_3_signal_control.py` 读取工作簿中的 `表3 (Table 3)`。
- 时间索引 `t = 0, ..., 59` 对应 07:00 至 08:58，每个单位为 2 分钟。
